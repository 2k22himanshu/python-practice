# Student Management System

A simple Student Management System built using Python.

## Features
- Add Student
- View Student Records
- Search Student
- Delete Student
- Store Data in File

## Technologies Used
- Python
- File Handling
- Lists
- Dictionaries
- Loops
- Conditional Statements

## Project Structure

student-management-system/
│
├── main.py
├── students.txt
└── README.md

## How to Run

1. Install Python
2. Run the following command:

python main.py
